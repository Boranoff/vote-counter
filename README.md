# vote-counter

Малък инструмент подпомагащ броящите бюлетини (и застъпници) в отчитането на хартиените бюлетини.

Написан е на JavaScript, работи без нужда от интернет (освен при първоначално зареждане на изборните данни). Без възможност от външна намеса, както и лесен за проверка на кода от всеки, дори с малко опит в програмирането.

Удобството е че отчитането става бързо, и когато няколко човека едновременно го ползват, има лесен сравнение за грешки. Инструмента поддържа цветно кодиране: в зависимост от отчетените номера на бюлетини и на преференции се прави контролна сума (показана в кръгче, горе в дясно), която отговаря на цветен код, който е видим през цялото време като рамка, така че дори един бърз поглед към устройството на хората до теб, и може да се разбере дали има допусната грешка при отчитането при някой.

Има страница със сумарната информация с гласове по партии и преференции (като се отчитат и невалидните бюлетини). (Отваря се като се натисне брой отчетени бюлетини горе в дясно)
Както и пълната история на въведените бюлетини - така че да може да се направи сравнение с останалите преброяващи за евентуални пропуски и грешки.

Необходимо е единствено да се направи Server endpoint за извличане на данните според номера на секцията:

Структура на върнатия json:


	{
		"СЕКЦИЯ" : "1234567890",
		"РАЙОН" : "... XX МИР",
		"ОБЩИНА/ГРАД" : "гр......",
		"АДРЕС" : "ГР....., ул./ж.к. .... №/бл....., вх. ...",
	  
		"parties" : {
			"0" : "Независим", 				    # 0 запазено за независими кандидати, ако има такива
			"-1": "Невалидна бюлетина", 	# -1 запазено за невалидни бюлетини
			"1" : "...", 					        # номер на партия според жребия : име на партия
			"2" : "..."
		},
	  
		"prefs" : {
			"0" : [                       #  номер на партия или 0 за независими кандидати, ако има такива
					{"101":"..."}, 			      # номер на преференция : име на кандидат
					{"102":"..."}
				  ],
			"1" : [							          # номер на партия или 0 за независими кандидати, ако има такива
					{"101":"..."},			      # номер на преференция : име на кандидат
					{"102":"..."}
				  ],
			"2" : [
					{"101":"..."},
					{"102":"..."}
				  ]
		}
	}
